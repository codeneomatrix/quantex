# Quantex

[Quantex](https://hex.pm/packages/quantex) is a quantum computer system environment and libraries in Elixir. Here is an example:

```elixir
iex> 

```

See the [online documentation](https://hexdocs.pm/quantex).

## Installation

Add to your ```mix.exs``` file:

```elixir
def deps do
  [
    { :quantex, "~> 0.0.1" }
  ]
end
```
